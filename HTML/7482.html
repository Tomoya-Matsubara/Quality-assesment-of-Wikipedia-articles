Triggertrap is a programmable camera trigger. It has an auxiliary input built around Arduino open-source architecture, meaning it can be programmed to trigger a camera in response to almost any electrical input like light, laser and sound1 2. The triggertrap is compatible with a large range of cameras, via cable or infra-red communication. It also has a built-in time-lapse function3 4

Background

The story of Triggertrap starts back in July 2011. The designers wanted to raise some capital for this new product. Therefore in an attempt to raise $25,000 for this project within over a month, nearly 900 supporters had pledged more than $77,000 in exchange for more than 950 Trigger traps via crowd-funding website kickstarter which is nearly three times more then what they wanted for the project. 5 6 7 8 9 10 Triggertrap has been designed by Ziah Fogel & Haje Jan Kamps 11 12 13 14

Technology overview

Trigger trap can trigger the camera's shutter based on light (when someone trips its laser beam), sound (when a loud clap is heard) and time (at varying time intervals) and can be triggered by anything capable of generating an electronic signal trigger in any other method the user can think of via an Aux port.15 16 The Triggertrap device is based on an Arduino and is an open-source hardware project for photographers and geeks alike.17 The trigger trap is able to respond and fire the external flash such that it would correctly sync at shutter speeds down to 1/640th of a second- that<U+2019>s a response time of less than 1.6 milliseconds.18 The light sensor can also be used for being used with a laser beam. The device has a built-in sensor which is a microphone that can trigger the camera whenever the triggertrap senses the sound.19 Trigger trap also has an auxiliary port, to which the user can connect anything as long as the user can find a way to create an electric signal. The device can control hundreds of cameras via infra-red or wired trigger system (depending upon cameras capability)20.

"Part of the excitement about the Triggertrap", says its inventor, Haje Jan Kamps, "is that I have no idea what people are going to use it for. It's an incredibly versatile piece of photography kit, and because it is so easy to hack, I'm expecting creativity to go off the charts."21

Types

There are two types of trigger traps22. A triggertrap version and the shield version. Both the versions run on Arduino-compatible software, which makes the barrier to modification very low indeed. The team is planning to release both the software and hardware under open-source licenses23. This allows use of the easy-to-learn Arduino programming language to add new functionality to the device - or alter the built-in features - and easily uploads the new firmware to the Triggertrap with a USB cable.24 25

The 'Shield' version of the Triggertrap takes hack ability to a completely new level. It is a device that is connected at the back of an Arduino rapid prototyping platform board. By choosing the shield version, the user gets the satisfaction of soldering his own Triggertrap . User can choose to build the Triggertrap as specified using step-by-step instructions, or use the circuit board to create his own versions of the Triggertrap26.
